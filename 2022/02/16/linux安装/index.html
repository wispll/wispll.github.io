3.bios 设置

开机按F2（不同的主板有不同快捷键），进入bios。

Boot -> Boot Mode    UEFI

   -> USB Boot    Enabled

Security -> Secure Boot    DIsabled

同时需要将USB作为第一启动项(放在设备列表的第一个)，或者开机的时候按F12选择启动方式。
